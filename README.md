# Создал несколько интересных сниппетов галерей на основе mixitup jquery plugin

Создал несколько сложных сниппетов на основе работы плагина mixitup jquery plugin. Они готовы к использованию в рабочих проектах. Чтобы код вставить, просто скопируйте нужный отрезок кода и вставьте его в соответсвующие файлы. Для простоты работы scss-код развелен по разным файлам
